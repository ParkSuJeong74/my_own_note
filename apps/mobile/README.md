# MY OWN NOTE MOBILE

**My Own Note MOBILE** 는 에디팅 웹/앱 서비스를 개발하는 개인 프로젝트의 AOS/IOS 프론트엔드 입니다.

## 기술 스택 (Tech Stack)

- **프레임워크**: React-Native Expo
- **라이브러리**:

## 초기 세팅

```bash
# React-Native Expo 세팅 (typescript)
npm i -g expo-cli
expo init .

# xcode는 App Store에서 설치 필요, xcode에서 ios 17.0도 설치해야 시뮬레이터 사용 가능
# android는 android-studio 설치 필요
brew install --cask android-studio

npm run aos
npm run ios
```
